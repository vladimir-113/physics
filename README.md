# NUKEMAP · Симулятор ядерного взрыва

Образовательный проект по физике: интерактивная визуализация физических последствий ядерного взрыва с реальной картой, конфигурируемой бомбой и сценариями доставки.

## Возможности

- **Конфигуратор бомбы**. Выбор делящегося материала (U-235, Pu-239), массы, схемы подрыва (пушечная / имплозия / термоядерная). Мощность считается из массы и КПД, а не задаётся напрямую.
- **Реальная карта** на Leaflet + OpenStreetMap. Зоны поражения рисуются поверх настоящего города. Можно ткнуть в любую точку или выбрать город из списка.
- **Точки наблюдения**. Поставь маркер на карте — увидишь, что произойдёт на этом расстоянии (выбьет ли окна, упадёт ли здание, время прихода ударной волны).
- **Сценарии сброса**: воздушный взрыв на оптимальной высоте, сброс с самолёта, боеголовка МБР, наземный взрыв, подземное испытание. Каждый сценарий по-своему влияет на радиусы поражения и осадки.
- **Кинематографичная анимация взрыва**: вспышка, ударная волна, расходящиеся зоны, грибовидное облако, тряска камеры.
- **Образовательная вкладка** с физикой деления, цепной реакции и термоядерного синтеза.
- **Адаптив**: работает на телефоне и компьютере.

## Запуск

Это статический сайт без сборки. Можно открыть прямо `index.html`, но для ES-модулей нужен HTTP-сервер. Самый простой способ:

```bash
# Python (есть везде)
python3 -m http.server 8000

# Или Node
npx serve .
```

Откроется на `http://localhost:8000`.

## Что под капотом

### Формулы

- **Мощность из массы**: `Y = N · E_fission`, где `N = (m / M) · N_A · η` — число прореагировавших атомов, `η` — КПД (0.015 для пушечной схемы U-235, до 0.20 для имплозии Pu-239).
- **Радиусы поражения**: классическое cube-root scaling из Glasstone & Dolan: `R = K · Y^(1/3)`, где коэффициенты K подобраны под уровни 20 psi / 5 psi / 1 psi для ударной волны, и под порог ожогов 3-й степени для теплового излучения.
- **Высота воздушного взрыва**: `H ≈ 200 · Y^(1/3)` метров — оптимум для максимальной площади ударной волны при минимуме осадков.

### Все цифры — упрощённые модели для образовательных целей. Это не симулятор для специалистов.
