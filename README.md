# FISSION-CTRL · Ядерный симулятор

Образовательный симулятор ядерного взрыва с реальной физикой, 3D-анимацией и интерактивной картой.

## Что внутри

- **Конструктор бомбы** — выбор типа реакции (деление / бустированное / термоядерное), схемы сборки (gun / implosion), делящегося материала (U-235 / Pu-239 / U-233), массы, обогащения, тампера и инициатора. Расчёт критической массы, k<sub>eff</sub>, КПД и выхода в реальном времени.
- **6 исторических образцов** — Little Boy, Fat Man, Царь-Бомба, B83, W88, РДС-1 с досье и историей.
- **Карта целей** — Leaflet + OpenStreetMap. Клик по карте ставит эпицентр, рисуются 7 концентрических зон поражения (огненный шар, 20/5/1 psi, ожоги 1–3 ст.).
- **3D-симуляция взрыва** на Three.js — вспышка, огненный шар, ударная волна, грибовидное облако с тороидальным вихрем, разрушение зданий по слоям избыточного давления, кольцо Вильсона.
- **Теория** — формулы и константы из Glasstone & Dolan "The Effects of Nuclear Weapons" (1977).

## Физика

Все формулы и величины из открытых источников. Симулятор даёт корректный **порядок величин**, а не точные секретные данные:

- Критические массы — из учебников (Sublette, Nuclear Weapon Archive).
- Радиус ударной волны — решение Сэдова–Тейлора: R(t) = ξ · (E·t²/ρ)^(1/5).
- Огненный шар: R<sub>fb</sub> ≈ 110 · Y^0.4 м.
- Высота гриба: H ≈ 6.5 · Y^0.35 км.
- Оптимальная высота подрыва: H<sub>opt</sub> ≈ 220 · Y^0.33 м.
- КПД (формула Бете): η ≈ (Δk · α)^(3/2).

Здесь нет ничего, чего нет в открытом учебнике физики или Википедии. Цель — показать **почему** ядерное оружие настолько опасно, на реальных числах.

## Запуск локально

Простой статический сайт. Достаточно открыть `index.html` через локальный сервер (через `file://` не заработает из-за ES-модулей):

```bash
python3 -m http.server 8000
# или
npx serve .
```

Затем открой http://localhost:8000

## Деплой на GitHub Pages

1. Создай новый репозиторий на GitHub.
2. Загрузи содержимое этой папки (через git push или drag-and-drop в web UI).
3. **Settings → Pages → Source: Deploy from a branch → main / root → Save**.
4. Через минуту сайт будет на `https://<твой-логин>.github.io/<репозиторий>/`.

## Зависимости (всё через CDN, ничего не нужно ставить)

- Three.js 0.160 — 3D рендеринг
- Leaflet 1.9.4 — карта
- OpenStreetMap + CartoDB Dark — тайлы карты
- Шрифты: JetBrains Mono, Major Mono Display (Google Fonts)

## Управление в 3D-вкладке

- **ЛКМ + drag** — вращение камеры
- **ПКМ + drag** — панорамирование
- **Колесо** — зум
- **↻ CAM** — переключение режимов (orbit / follow / top / ground)
- **► 1×** — скорость симуляции (0.5× / 1× / 2× / 4×)

## Технические нюансы

- Single-page приложение, два файла: `index.html` + `app.js`.
- Никаких бэкендов и сборщиков. Просто открой в браузере.
- Для поиска городов используется бесплатное Nominatim API (OpenStreetMap). Без ключей.

## Лицензия / отказ от ответственности

Образовательный проект. Создан, чтобы наглядно показать масштаб разрушений от ядерного оружия и почему весь мир работает над его нераспространением. Никаких практически применимых сведений о конструировании оружия здесь нет.
